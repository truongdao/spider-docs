report.js
Thư viện hỗ trợ việc xuất các báo cáo text file  (giống một trang jsp nhưng dùng js làm ngôn ngữ nền).
Có thể truyền dữ liệu vào cho report dưới dạng json, nếu gọi từ chương trình javascript, chỉ đơn giản là xuất đối tượng dữ liệu đó ra thành dạng chuỗi, và truyền vào hàm xử lý report, hàm xử lý report sẽ đặt dữ liệu đó vào biến toàn cục tên 'input', trang báo cáo cần truy xuất biến này để lấy dữ liệu ra. Trang report sẽ được xử lý trên một script engine riêng nên nó không thể trực tiếp truy xuất các biến của chương trình.
Có thể thay đổi các ký tự đánh dấu embedded code thay vì <%%> như thông lệ bằng cách ghi đè biến INLINE_OPEN, INLINE_CLOSE trước khi chạy. Việc này có ý nghỉa là ta có thể tận dụng các ký tự comment của trang gốc để chứa code xử lý. Ví dụ nhúng vào C, ta có thể dùng cặp  {INLINE_OPEN, INLINE_CLOSE} = { "/*!", "!*/"} để nhận diện.

Example: 

 
	----- test1.jrt file -----
	 
	  <p>Counting to three:</p>
		<% for (var i=1; i<=input.lines; i++) { %>
		 <p>This number is <%= i %>.</p>
		<% } %>
		<p><%=input.name%></p>
	--------------------------------------
> using report lib (java): 
  
	String report_out = ProcessJRT.executeJRT("test1.jrt", "{lines:3, name:"Truong"}");
	print(report_out);
> using (js) by import report.js 
  
	report_out = report.executeJRT("test1.jrt", "{lines:3, name:"Truong"}");
	outln(report_out);
> output: 
 
	<p>Counting to three:</p>
	<p>This number  is 1.</p>
	<p>This number  is 2.</p>
	<p> Truong </p>
